## 🎯 ЩО ВКЛЮЧАЄ КАРТА САЙТУ

### **Розділ 1: Категорії продуктів**
- Вали тиснення (Каландри) з типами та застосуванням
- Анілоксові вали та Формні гільзи з параметрами
- Обладнання та Комплектуючи (7 категорій)

### **Розділ 2: Структура веб-сайту**
- Головна сторінка (Hero, About, CTA)
- Каталог з трьома категоріями
- Пошук та фільтри
- Прайс-лист
- Технічна документація
- Контактна форма
- Footer

### **Розділ 3: Функціональні вимоги**
- Таблиця всіх функцій (пошук, фільтри, карточки, прайс, документи, контакти, адаптивність)

### **Розділ 4: Дизайнові рекомендації**
- Палітра: Синій (#003D82), Помаранчевий (#FF6B35)
- Типографія: Roboto, Inter
- Адаптивність: Desktop, Планшет, Мобіль (iOS Safari)

### **Розділ 5: Технічна архітектура**
- Backend: Django 5+, Python 3.12+, PostgreSQL
- Frontend: HTMX, Vanilla JS, CSS BEM
- Deploy: Docker + DigitalOcean

### **Розділ 6: Графік розробки**
- Фаза 1 (1–2 дні): Архітектура
- Фаза 2 (3–4 дні): Frontend
- Фаза 3 (1 день): Контакти
- Фаза 4 (1–2 дні): Оптимізація та Deploy
- **ЗАГАЛОМ: 6–9 днів**
