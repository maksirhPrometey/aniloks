## 🎯 СЕГМЕНТИ ПРОДУКТІВ / ПОСЛУГ

Компанія спеціалізується на 3 основних категоріях обладнання та матеріалів:

### 1. **Вали тиснення (Каландри)**
   - Металеві циліндри для видавлювання текстури на матеріалах
   - Типи: Сталь по сталі, Сталь по гумі, Сталь по паперу
   - Застосування: папір, картон, полімери, шкіра, метали, текстиль

### 2. **Анілоксові керамічні вали та Формні гільзи**
   - Вали для переносу фарби (флексографія)
   - Формні гільзи (sleeves) для наклейки флексоформ
   - Параметри підбору: лініатура, об'єм осередків, форма осередків

### 3. **Обладнання та Комплектуючи**
   - Щітки для чищення анілоксових валів
   - Ракельні ножі
   - Електронні навантажувачі (1000–1500 кг)
   - Машина ультразвукового миття
   - Машина склеювання та намотування рукава
   - Пакеторобні машини (майка, фасування, Wicket, Doy-pack)
   - Флексографічні машини (ярусні, планетарні, лінійні)

## 📋 ФУНКЦІОНАЛЬНІ ВИМОГИ

### 🔍 Пошук і Фільтрація
- **Глобальний пошук** по: назві, артиклу, описі, матеріалу, параметрам
- **Фільтри за категоріями** (3 основні)
- **Фільтри за параметрами:**
  - Для валів: розмір, матеріал, кількість
  - Для анілоксів: лініатура, об'єм, тип фарби
  - Для обладнання: потужність, швидкість, вага, вантажопідйомність
- **Сортування:** назва (A-Z), популярність, новизна, ціна
- **Збереження фільтрів** у URL параметрах

### 📄 Карточки Продуктів
- Назва, артикль, зображення (основне + галерея)
- Короткий опис
- Основні параметри (таблиця)
- Повний опис з гіпер-специфікаціями
- Матеріали, що обробляються (для обладнання)
- Версії / модифікації товару (якщо вони є)
- CTA: "Запит комерційної пропозиції", "Додати в кошик" (якщо буде е-комерс)
- Посилання на техдокументацію (PDF)
- Опціонально: Фото / відео продукту в роботі

### 💰 Прайс-лист
- Таблиця: Артикль | Назва | Опис | Вартість | Валюта (USD, EUR, UAH)
- Форма при наведенні: "Запросити актуальну ціну" (оскільки ціни динамічні)
- Скачування у форматі XLSX, PDF
- Секція "Гуртові знижки" (якщо є)

### 📥 Технічна Документація
- Скачуванням каталогів у PDF за категоріями
- Документи відсортовані за розділами (Вали, Анілокси, Обладнання)
- Опціональне вбудовування переглядача PDF на сторінці
- Версіонування (версія каталогу, дата оновлення)

### 📞 Контактна Форма
- **Поля:**
  - ПІБ, назва компанії, посада
  - Email, телефон
  - Категорія запиту (Консультація, Замовлення, Сервіс, Рекламація)
  - Текст повідомлення
  - Опціональне прикріплення файлів (техзавдання, кресленики)
- **Обробка:** Email на nikolajcornejko@gmail.com
- **Підтвердження:** Автоматична відповідь з номером заявки
- **Бекенд:** CRM інтеграція (якщо є; можна почати з простого емейлу)

## 🎨 ДИЗАЙНОВІ ВИМОГИ

- **Заголовки:** Сучасний sans-serif (e.g., Roboto, Inter)
- **Тіло:** Комфортна для читання sans-serif
- **Розміри:** Ієрархія від H1 до H6, P

### Адаптивність
- **Desktop:** 1440px+, 2-3 колони
- **Планшет:** 768px–1439px, 1-2 колони, складання меню
- **Мобіль iOS/Android:** 320px–767px
  - Повна мобільна адаптація (дотикові елементи, вертикальний скрол)
  - iOS Safari: тестування на панелі нав'ігації, viewport

## 🛠️ ТЕХНІЧНА АРХІТЕКТУРА

### Backend
- **Framework:** Django 5+
- **Python:** 3.12+
- **БД:** PostgreSQL
- **Структура:** `src/` з підпроектами (products, catalog, contact, docs)

### Frontend
- **JS:** HTMX + Vanilla JS (без React/Vue)
- **CSS:** BEM + CSS custom properties (без !important)
- **Бандлер:** Ні (статичні файли через nginx)

### Статика (Production)
- `nginx` + `collectstatic`
- `/static/` директорія для CSS, JS, зображень
- `/media/` для завантажень (PDF, каталоги, зображення продуктів)

### Конфіг
- `python-decouple` для `.env`
- Сплит settings (base, develop, production, test)

### Deploy
- **DigitalOcean** + Docker + nginx + PostgreSQL
- Dockerhub образи
